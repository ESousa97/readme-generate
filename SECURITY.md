# Security Policy

## Repository status

This repository is archived and no longer actively maintained.
It remains public for study purposes only.
There is no guarantee of response, review, fix, or patch releases.

## Supported Versions

| Version | Supported |
| --- | --- |
| 1.x | yes |
| < 1.0 | no |

## Reporting a Vulnerability

Please report vulnerabilities privately using one of the channels below:
- GitHub Security Advisory: https://github.com/enoquesousa/readme-generate/security/advisories/new
- Portfolio contact: https://enoquesousa.vercel.app

Do not open public issues for security vulnerabilities.

## Response SLA

- No guaranteed SLA for acknowledgment, triage, or mitigation.

## Disclosure Policy

- We follow responsible disclosure.
- A fix is prepared and released before full public details.
- CVE assignment is considered when applicable.
